Denial of Service


is a type of attack that is done too stop the victim to access information or resources. the most normal type of this attack is DDoS which is distributed denial of service, this uses multiple ip addresses to overwhelm the resource and slow down its ability to answer request to the point of not working. The reason this attack is so hard to stop is because it is hard to see the difference between a normal user and a DoS address, these ip addresses could also be part of a botnet

Botnet

is a network of ip addresses which originally belongs to people who dont know they are a part of the botnet, but the one who created the botnet has gotten access to the device.
